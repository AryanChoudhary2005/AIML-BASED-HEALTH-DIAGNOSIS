# AIML-Based Health Diagnosis System

Developed an AI-powered healthcare platform designed to assist in the early detection of multiple diseases through Machine Learning and Deep Learning models. The project integrates predictive analytics, medical image classification, and an intuitive web interface to provide fast and accessible health assessments.

### Key Features

* Early detection of **Diabetes, Heart Disease, Pneumonia, and Parkinson's Disease**
* AI-based disease prediction using structured health data and medical images
* Interactive web application for real-time predictions
* Responsive frontend with seamless user experience
* Cloud deployment for online accessibility

### Technologies Used

**Python, Scikit-learn, TensorFlow/Keras, CNN, Logistic Regression, Random Forest, SVM, Streamlit, Vite, HTML, CSS, JavaScript, Vercel**

### Project Highlights

* Live link: https://ai-ml-based-health-diagnosis-vpxp-ag7j9jcfv-rico24251s-projects.vercel.app/
* Built and trained Machine Learning models for diabetes, heart disease, and Parkinson's prediction using clinical datasets.
* Developed a Convolutional Neural Network (CNN) for pneumonia detection from chest X-ray images with image preprocessing and data augmentation techniques.
* Performed data cleaning, preprocessing, feature scaling, and model evaluation to improve prediction performance.
* Integrated trained models into a user-friendly web application using Streamlit and later enhanced the frontend using Vite for better performance and responsiveness.
* Deployed the application on Vercel, enabling users to upload medical data or X-ray images and receive instant AI-driven predictions.

### Results

* Diabetes Prediction Accuracy: **93.4%**
* Pneumonia Detection Accuracy: **91.6%**
* Heart Disease Prediction Accuracy: **89.3%**
* Parkinson's Disease Prediction Accuracy: **87.2%**

This project strengthened my understanding of end-to-end AI application development, including data preprocessing, model training, deep learning, web integration, deployment, and real-world healthcare applications. It also involved preparing a complete research paper covering literature review, methodology, system architecture, experimental results, and future scope.
